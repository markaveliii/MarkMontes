Personal Projects

Level-JJAMM

Level-JJAMM is a dungeon based speed-run type of game. It is a game that uses the curses library to make a cellular 2D environment. The objective of the game is to clear each level, where each level presents objectives such as moving enemies, find the key, or obtain the weapon. You can die as many times as you want in the game, but it records each of your deaths. It also records you time and gives you those values at the end.
